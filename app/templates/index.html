POST /users
  • Authentication: None
  • Example Minimum Payload: { "firstName": "Ash", "lastName": "Ketchum", "username": "ashk", "email": "ashk@mastertrainer.org", "password": "pika123" }

  • Authentication: None
  • Example Response: { "ingredients": [ { "ingredient_id": 1, "ingredient": "Tomato", "description": "Fresh and juicy red tomatoes", "status": false, "dateAdded": "2022-04-20" }, { "ingredient_id": 2, "ingredient": "Onion", "description": "Crunchy and flavorful onions", "status": true, "dateAdded": "2022-04-21" } ] }

  • Authentication: None
